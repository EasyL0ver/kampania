# Meta Instructions

These rules apply to all work on this campaign. Always follow them.

1. **Historical facts → `historical context\`** — Whenever a new historical fact is learned or researched, add it to the appropriate file in the `historical context\` folder (or create a new file if it doesn't fit existing ones).

2. **Search in Polish** — When searching for sources and historical information, always search in Polish for better accuracy and depth of local sources.

3. **Fact-check player input** — When the user provides historical tidbits or details for the story, verify them for accuracy. Flag any inaccuracies found — small liberties are fine for the story, but always make the user aware of what's real and what's bent.

4. **Structured lore storage** — Background story elements (the story players uncover) are stored in dedicated folders:
   - `characters\` — One `.md` file per character (NPCs, historical figures, etc.)
   - `places\` — One `.md` file per location
   - `story-facts\` — One `.md` file per story fact / plot element / clue

5. **Placeholder naming** — Until a character or place has a final name, use `%PLACEHOLDER%` syntax (e.g. `%COLONEL%`, `%OLD_VILLAGE%`). This makes it easy to find-and-replace across all files later. **Never swap placeholders for real names unless explicitly told to by the user.**

6. **Inspirations → `inspirations\`** — Whenever a new inspiration is discussed (films, books, music, real events, vibes, references), put it into the `inspirations\` folder.
